Tablas en HTML

Pra poder usar las tablas en Html necesitaremos las etiquetas

  • table: Esta etiqueta abre la tabla
  • tr: Esta etiqueta abre una fila en la tabla
  • td: Esta etiqueta abre una columna (celda) en la fila
  • th: Esta etiqueta abre una columna (celda) de tipo cabecera en la fila
  • caption: Esta etiqueta abre la cabecera de la tabla
  • tfoot: Esta etiqueta abre la pie de la tabla
  • thead: Esta etiqueta abre la cabeza de la tabla

Para poder ver el desieno de la tabla necesitaremos usar de los atributos de la etiqueta de la tabla los cuales son los siguientes

  • border: Este atributo establece el ancho de las lineas que se van a poner en las celdas
  • colspan: define el tamaño de la coluna (Celdas)
  • cellpadding: Este atributo establece el espacio entre las celdas y las lineas
  • cellspacing: Este atributo establece el espacio entre las celdas
  • width: Este atributo establece el ancho de la tabla
  • height: Este atributo establece el alto de la tabla
  • align: Este atributo establece el alineamiento de la tabla en la pantalla
