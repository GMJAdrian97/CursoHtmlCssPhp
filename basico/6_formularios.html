Formularios en HTML

Los furmularoips en html son muy importantes ya que estos se encargarn de enviar texto, informacion al servidor o cualquier cosa

Para usar un formulario en html necesitamos la etiqueta form

Como sabemos los formularios tiene campos y estos se usan mediante las siguientes etiquetas que son las mas cumunes dentro de un formlario


  • input: Este es el elemento principal para crear campos de entrada, este campo tiene atributos importantes tales como
    • type: Este es el tipo de campo de entrada que deseamos crear (text, number, passwors, etc)
    • name: Este es el nombre que identifica el campo de entrada
    • value: Este es el valor que se muestra en el campo de entrada
    • placeholder: Este es el texto que se muestra en el campo de entrada cuando este esta vacio
    • required: Este es un atributo que indica que el campo es obligatorio
    • disabled: Este es un atributo que indica que el campo de entrada no se puede editar
  • Label Este es el nombre de los campos de entrada, este campo tiene atributos importantes tales como
    • for: Este es el nombre del campo de entrada que se relaciona con este label
  • textarea: Este es un campo de entrada para escribir varias lineas de texto, este campo tiene atributos importantes tales como
    • name: Este es el nombre que identifica el campo de entrada
    • rows: Este es el numero de filas que se mostraran en el textarea
    • cols: Este es el numero de columnas que se mostraran en el textarea
  • select: Este es un campo de entrada para seleccionar un item de una lista desplegable, este campo tiene atributos importantes tales como
    • name: Este es el nombre que identifica el campo de entrada
    • multiple: Este es un atributo que indica que el campo de entrada puede seleccionar varios items
    • size: Este es el numero de items que se mostraran en la lista desplegable
    • required: Este es un atributo que indica que el campo es obligatorio
  • radio: Este es un campo de entrada para seleccionar una opcion de una lista de seleccion multiple
  • checkbox: Este es un campo de entrada para seleccionar opcion multiple
  • submit: Este es un boton para enviar los datos del formulario
